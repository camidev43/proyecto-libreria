# Checkbox

El componente `Checkbox` permite seleccionar una o varias opciones de una lista. Es altamente personalizable y soporta diferentes variantes visuales, límites de selección, estados deshabilitados, requeridos, personalizados y más.

## Accesibilidad

- Usa roles y atributos ARIA apropiados.
- Soporta estados deshabilitado, requerido y solo lectura.
- Provee siempre un `label` para accesibilidad.

## Ejemplo de opciones

```ts
const opciones: Option[] = [
	{ value: 'uno', label: 'Uno', descripcion: 'Primera opción' },
	{ value: 'dos', label: 'Dos', descripcion: 'Segunda opción' },
	{ value: 'tres', label: 'Tres', descripcion: 'Tercera opción' },
];
```

## Personalización

- `relleno`, `color`, `borderColor`, `size`, `radius` para estilos.
- `maxSelecionados` para limitar selección.
- `tachado` para tachar texto al seleccionar.
- `readOnly`, `disabled`, `required` para estados.

## Buenas prácticas

- Usa `CheckboxGroup` para la mayoría de los casos.
- Usa `CheckboxGroupCustom` para renderizados avanzados.
- Provee siempre un `label` para accesibilidad.
